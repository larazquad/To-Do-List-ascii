# To-Do List Responsiva

Uma aplicação web moderna e elegante para gerenciamento de tarefas diárias, desenvolvida com HTML, CSS e JavaScript puro.

## Características

- **Design Responsivo**: Adapta-se perfeitamente a dispositivos móveis, tablets e desktop
- **Interface Intuitiva**: Design limpo com gradientes e animações suaves
- **Funcionalidades Completas**: Adicionar, marcar como concluída e remover tarefas
- **Estatísticas em Tempo Real**: Acompanhamento de tarefas totais, concluídas e pendentes
- **Experiência Otimizada**: Efeitos visuais, suporte a dark mode e atalhos de teclado

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Media Queries, Animações)
- JavaScript Vanilla

## Como Usar

1. Digite uma tarefa no campo de texto
2. Clique em "Adicionar Tarefa" ou pressione Enter
3. Clique no texto para marcar/desmarcar como concluída
4. Use "Remover" para excluir tarefas individuais

Ideal para organização pessoal e produtividade no dia a dia.
